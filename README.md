## 📂 Проекты

### 📊 Анализ продаж (Power BI)

Разработан интерактивный дашборд для анализа продаж, позволяющий выявлять ключевые драйверы выручки и убыточные зоны бизнеса.

**Что сделано:**
- Построены KPI: выручка, прибыль, маржа, средний чек
- Проведён анализ по сегментам, регионам и категориям
- Выявлены убыточные заказы и проблемные зоны

**Результат:**
- Найдены ключевые источники выручки
- Обнаружены зоны снижения прибыли
- Сформулированы рекомендации для бизнеса

### 🧠 Анализ метрик (SQL + DataLens)

Проведён анализ ключевых метрик сервиса доставки еды с использованием SQL и визуализации в DataLens.

**Что сделано:**

**Результат:**
